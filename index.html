How I Think

PHILOSOPHY

I value clarity over cleverness. Code is read 10x more than it is written. I avoid "magic" abstractions that hide the underlying logic, because when they break, they are impossible to debug.

THE STACK

React + TypeScript + Node. It’s the "Honda Civic" of the web—reliable, the parts are everywhere, and it handles 95% of use cases perfectly. I only reach for complex tools when the scale demands it.

ANTI-PATTERNS

I avoid premature optimization. I build it to work first, then I measure it, then I make it fast. Optimizing without data is just guessing.
